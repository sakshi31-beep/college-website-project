Project Name: College Website

Developed By: Sakshi Nayadu
Course: Diploma in Computer Science Engineering

Software Requirements:
- XAMPP
- PHP
- MySQL
- Web Browser

How to Run the Project:
1. Install XAMPP.
2. Copy the project folder into xampp/htdocs/.
3. Open phpMyAdmin.
4. Create a database named ggp_raipur.
5. Import the file database/college.sql (or ggp_raipur.sql).
6. Open the browser and run:
   http://localhost/college_website/

Admin Panel:
http://localhost/college_website/admin/

Database File Location:
database/college.sql
